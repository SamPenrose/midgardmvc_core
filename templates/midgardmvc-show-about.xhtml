Midgard MVC is an application framework for the PHP programming language. It runs on top of Midgard2, a Free Software persistent storage platform for interactive web application development.

The first version of the Midgard platform was released in 1999, and it has been developed and maintained by an international community since then. First usable versions of Midgard MVC framework surfaced in 2008.
